# exp cafe' — เว็บไซต์ 7 หน้า + หลังบ้าน Google Sheets

## โครงไฟล์

```
index.html  about.html  menu.html  news.html
gallery.html  faq.html  contact.html      ← 7 หน้าตาม brief
assets/style.css   assets/app.js          ← ธีมและตัวขับเคลื่อน
data/content.json                         ← ข้อมูลสำรอง (ถ้าชีตล่ม เว็บยังไม่ขาว)
img/                                      ← รูปทั้งหมด ชื่อไฟล์ต้องตรงกับในชีต
apps-script/Code.gs                       ← หลังบ้าน
sheets/*.csv                              ← นำเข้าชีต 6 แผ่นได้ทันที
sitemap.xml  robots.txt                   ← SEO พื้นฐาน
build.py                                  ← สร้าง 7 หน้าใหม่เมื่อแก้ nav/footer
```

## ติดตั้ง (ครั้งเดียว)

1. สร้าง Google Sheet ใหม่ในบัญชี Gmail ของร้าน
2. นำเข้า CSV ทีละไฟล์จาก `sheets/` → ไฟล์ > นำเข้า > แทนที่แผ่นงานปัจจุบัน
   (หรือรัน `setupSheet()` ให้สร้างหัวคอลัมน์เปล่าแล้วพิมพ์เอง)
3. ส่วนขยาย > Apps Script > วาง `apps-script/Code.gs` > ใส่ `SHEET_ID`
4. Deploy > New deployment > Web app · Execute as **Me** · Access **Anyone**
5. คัดลอก URL ที่ลงท้าย `/exec` ไปวางใน `CONFIG.endpoint` ของ `assets/app.js`
6. อัปโหลดรูปทั้งหมดลง `img/` (ชื่อไฟล์ตรงกับคอลัมน์ `image` ในชีต)
7. ขึ้น GitHub Pages > ทดสอบเปิดทุกหน้าบนมือถือ 1 รอบ

## ลูกค้าแก้เนื้อหาเองยังไง

แก้ในชีตแล้วรีเฟรชเว็บ — อัปเดตเองภายใน 5 นาที
อยากให้ขึ้นทันที: ในชีตกดเมนู **เว็บไซต์ร้าน > อัปเดตเว็บทันที**

| แผ่น | แก้อะไรได้ |
|---|---|
| `site` | ชื่อร้าน สโลแกน เรื่องราว เบอร์ ไลน์ เฟซบุ๊ก ที่อยู่ |
| `hours` | เวลาเปิด–ปิดรายวัน (พิมพ์ `ปิด` ในวันหยุด) |
| `menu` | เมนู ราคา ป้ายแนะนำ |
| `news` | ข่าว/อีเวนต์ |
| `gallery` | รูปบรรยากาศ |
| `faq` | คำถาม–คำตอบ |

คอลัมน์ `show` พิมพ์ `FALSE` = ซ่อนรายการนั้นจากเว็บ โดยไม่ต้องลบแถวทิ้ง

## ยังค้างอยู่

- [ ] แปลงรูป `.jfif` → `.webp` (Safari/iPhone เปิด `.jfif` ไม่ขึ้น)
- [ ] เลือกจังหวัดให้ตรงกัน — ที่อยู่ระบุเชียงใหม่ แต่คีย์เวิร์ด SEO เป็นอุดรธานี
- [ ] `seoDesc` ยังว่าง (ตอนนี้ระบบใช้ `aboutShort` แทนไปก่อน)
- [ ] คำบรรยายรูปแกลเลอรี 3 รูปยังว่าง
- [ ] `img/og.jpg` (ภาพตอนแชร์ลิงก์) และ `img/favicon.png`
- [ ] ราคา 0 บาท และเบอร์ 0999999999 — ข้อมูลทดสอบ ต้องเปลี่ยนก่อนขึ้นจริง

หมายเหตุ: ทำ SEO พื้นฐานให้ครบ (meta, Open Graph, JSON-LD ประเภทร้านกาแฟ, sitemap,
robots) แต่ **ไม่รับประกันอันดับบน Google** อันดับขึ้นกับหลายปัจจัยที่ควบคุมไม่ได้
